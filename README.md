This project is a **Streamlit dashboard** designed for pharmacies to monitor sales, profit, and stock expiry risk. It helps make data-driven decisions and optimize inventory management.

## Features

- Display total medicines and average profit.
- Top 10 profitable medicines.
- Highlight high expiry risk medicines.
- Filters by manufacturer, stock movement, and expiry risk.
- Interactive charts for profit distribution and stock analysis.
- Exportable PDF reports for client insights.
- Ready for local use or deployment on Streamlit Community Cloud.

## Dataset

The project uses pharmacy retail data including:

- Medicine name, composition, uses, side effects
- Manufacturer
- Review percentages
- Stock and profit data

The dataset is included as `outputs/retail_pharmacy_analytics.csv`.
